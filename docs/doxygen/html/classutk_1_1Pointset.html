Detailed Description

template<unsigned int D, typename T, typename P>
class utk::Pointset< D, T, P >

A vector of D-dimensional samples with coordinates of type T.

This class is used to represent a point set. It is at the core of the UTK framework and is voluntarily kept as simple as possible and heavily templated for the sake of code modularity.
