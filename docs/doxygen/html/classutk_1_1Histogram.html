Detailed Description

template<uint D, typename T0, typename T1>
class utk::Histogram< D, T0, T1 >

A D-dimensional histogram.

This class is mostly used to represent the PCF of a sampling pattern (1D histogram), its fourier spectrum (2D histogram), or its spectral radial average (1D histogram).
